# Two Pointers Technique

## Explanation
The Two Pointers technique involves using two integer indices to traverse a data structure (usually an array or linked list) simultaneously.

## Common Patterns
- **Opposite Ends**: Pointers start at the beginning and end, moving towards each other (e.g., reversing an array, 2SUM in a sorted array).
- **Same Direction**: Pointers move at different speeds or start at different positions but move in the same direction (e.g., removing duplicates, finding a sub-array).

## Complexity Analysis
- **Time Complexity**: O(n) - Since each pointer typically traverses the data structure at most once.
- **Space Complexity**: O(1) - Only a few extra variables are used for the pointers.
