# Backtracking

## Explanation
Backtracking is an algorithmic technique for solving problems recursively by trying to build a solution incrementally, one piece at a time, removing those solutions that fail to satisfy the constraints of the problem at any point in time.

## Examples
- N-Queens Problem
- Sudoku Solver
- Generating Permutations/Subsets

## Complexity Analysis
- **Time Complexity**: Generally exponential, e.g., O(k^n) where n is the number of decisions and k is the number of choices per decision.
- **Space Complexity**: O(n) - The depth of the recursion tree.
