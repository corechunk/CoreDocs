# Greedy Algorithms

## Explanation
A Greedy algorithm makes the locally optimal choice at each step with the hope of finding a global optimum.

## Examples
- Huffman Coding
- Kruskal’s and Prim’s Minimum Spanning Tree algorithms
- Dijkstra’s Shortest Path algorithm
- Fractional Knapsack Problem

## Complexity Analysis
- **Time Complexity**: Varies greatly; often O(n log n) due to initial sorting.
- **Space Complexity**: Varies, often O(1) or O(n).
