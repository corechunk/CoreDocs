# Prefix Sums & Difference Arrays

## Explanation
- **Prefix Sum**: Precomputing the sum of elements from the start to each index. This allows for O(1) range sum queries.
- **Difference Array**: Storing the difference between adjacent elements. This allows for O(1) range updates.

## Complexity Analysis
- **Prefix Sum Build**: O(n) time, O(n) space.
- **Prefix Sum Query**: O(1) time.
- **Difference Array Build**: O(n) time, O(n) space.
- **Difference Array Update**: O(1) time.
