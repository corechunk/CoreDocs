# Graph Traversals (BFS & DFS)

## Explanation
Methods for visiting all the vertices of a graph in a specific order.

## Algorithms
- **Breadth-First Search (BFS)**: Visits all neighbors of a vertex before moving to the next level. Uses a Queue.
- **Depth-First Search (DFS)**: Explores as far as possible along each branch before backtracking. Uses a Stack or Recursion.

## Complexity Analysis
- **Time Complexity**: O(V + E) - where V is the number of vertices and E is the number of edges.
- **Space Complexity**: O(V) - to store the visited status and the queue/stack.
