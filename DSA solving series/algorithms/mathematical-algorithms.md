# Mathematical Algorithms

## Explanation
Various mathematical techniques and algorithms used in competitive programming and computer science.

## Topics
- **Sieve of Eratosthenes**: Efficiently finding all primes up to a limit.
- **Greatest Common Divisor (GCD)**: Using the Euclidean algorithm.
- **Fast Exponentiation**: Computing `a^b % m` in logarithmic time.
- **Convex Hull**: Finding the smallest convex polygon that contains all points.

## Complexity Analysis
- **Sieve**: O(n log log n) time, O(n) space.
- **GCD**: O(log(min(a, b))) time.
- **Fast Exponentiation**: O(log b) time.
- **Convex Hull (Graham Scan)**: O(n log n).
