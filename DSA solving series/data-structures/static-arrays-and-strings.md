# Static Arrays & Strings

## Explanation
Static Arrays are collections of elements of the same type stored in contiguous memory locations. The size is fixed at the time of creation and cannot be changed. Strings are essentially arrays of characters, often terminated by a null character or managed with a length property.

## Common Methods/Actions
- **Access**: O(1) - Retrieve an element at a specific index.
- **Update**: O(1) - Change the value of an element at a specific index.
- **Traverse**: O(n) - Iterate through all elements.
- **String Concatenation**: O(n + m) - Joining two strings.
- **Substring Extraction**: O(k) - Getting a part of a string.
