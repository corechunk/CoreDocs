# 📊 Bottom-Up LALR(1) & LR(k) Parser Generators

## 1. Shift-Reduce Engine
Bottom-up parsers construct syntax trees from leaf nodes up to the root symbol using state tables (generated by tools like Yacc, Bison, or ANTLR).

- **Shift:** Reads incoming lexer tokens and pushes them onto an internal state stack.
- **Reduce:** Replaces a sequence of matching right-hand grammar symbols on the stack with their corresponding left-hand non-terminal symbol.

## 2. Shift/Reduce Conflicts
Occurs when the parser state table cannot uniquely decide whether to shift the next incoming token or perform a grammar reduction based on a single lookahead token.
