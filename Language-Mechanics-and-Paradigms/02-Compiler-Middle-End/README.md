# ⚙️ Phase 02: Compiler Middle-End

> ### 🛠️ Required Knowledge Depth & Requirements Coverage
> * **Semantic Validation:** Type resolution passes, implicit coercion verification, symbol resolution across scope bounds, immutability & const correctness enforcement.
> * **Intermediate Representations:** Three-Address Code (TAC) quads/triples, Static Single Assignment (SSA) form (Phi nodes, dominance frontiers), Control Flow Graphs (CFGs).
> * **Optimization Heuristics:** Compile-time constant folding & propagation, Dead code elimination (unreachable basic blocks), Function inlining & loop unrolling.
