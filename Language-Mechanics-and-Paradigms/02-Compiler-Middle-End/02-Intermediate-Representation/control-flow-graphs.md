# 🕸️ Control Flow Graphs (CFG) & Basic Blocks

## 1. Basic Blocks
A Basic Block is a linear sequence of instructions with exactly one entry point at the top and one exit point at the bottom (no internal jumps or branches).

## 2. Graph Construction & Branch Edges
Compilers connect Basic Blocks into a **Control Flow Graph (CFG)** using directed edges representing jumps, conditional branches, and loop feedback paths.
