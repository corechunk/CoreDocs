# 🔀 Phase 07: Hybrid Unified Pipeline

> ### 🛠️ Required Knowledge Depth & Requirements Coverage
> * **Multi-Stage Engine Integration:** Seamlessly routing CLI interactive shell commands vs script AST evaluation within a single unified scope environment.
> * **Dynamic Promotion Hooks:** On-Stack Replacement (OSR) to swap slow active bytecode stack frames directly into compiled native JIT machine code at runtime.
> * **Static Binary Export:** Compiling active JIT memory pages down into standalone, executable static ELF disk binaries.
> * **Embedded Systems & Network Bindings:** Integrating low-level socket APIs (`socket`, `bind`, `listen`), threading models, and JSON serialization directly into the engine's standard library.
