# 🧬 Type Inference and Generics

## 1. Type Inference (Hindley-Milner)
Allows compilers to infer variable types automatically without explicit developer annotations while maintaining strict static type safety.

## 2. Parametric Polymorphism (Generics / Traits)
Enables routines and data structures to operate over abstract type placeholders, compiled via monomorphization (C++ templates / Rust generics) or boxing (Java generics).
