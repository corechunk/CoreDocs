# 🏷️ Module 03: Type Systems

This module explores type system architecture, static vs. dynamic checking, and structural boundaries.

---

## 📄 Core Topics

- **[Static vs. Dynamic Typing](./static-vs-dynamic-typing.md):** Compile-time validation vs. runtime dynamic evaluation, type inferencing, and mutability (`var`/`val`).
- **[Nominal vs. Structural Typing](./nominal-vs-structural.md):** Explicit class naming vs. shape-based interface validation (OOP vs. Plain Struct bounds).
- **[Type Inference & Generics](./inference-and-generics.md):** Hindley-Milner algorithms, parametric polymorphism, and trait bounds.
