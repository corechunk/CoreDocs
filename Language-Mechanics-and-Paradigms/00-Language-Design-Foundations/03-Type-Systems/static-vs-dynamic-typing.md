# 🏷️ Static vs. Dynamic Typing Mechanics

## 1. Type Enforcement Spectrum
- **Static Typing (C, C++, Rust, Kotlin):** Variable types are bound and validated at compile time.
- **Dynamic Typing (Python, JavaScript, Lua):** Types are associated with runtime values rather than variable identifiers.

## 2. Mutability Annotations: `var` vs `val`
- `val` (Immutable): Once initialized, the reference or value allocation cannot be reassigned.
- `var` (Mutable): The identifier allows re-assignment to new values matching its declared type.
