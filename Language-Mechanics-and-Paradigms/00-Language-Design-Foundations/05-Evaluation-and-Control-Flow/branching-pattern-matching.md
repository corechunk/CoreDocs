# 🔀 Branching & Pattern Matching

## 1. Mandatory Base Control Flow
Base language engines require core control primitives:
- **Conditionals:** `if`, `else if`, `else`
- **Loops:** `while`, `for`
- **Exits:** `return`

## 2. Advanced Pattern Matching Engines
Advanced execution modules compile `match` constructs down to high-performance jump tables or decision trees.
