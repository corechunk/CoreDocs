# 📜 Official Documentation Index (The Sources of Truth)

This is a high-signal index linking directly to the definitive official documentation, standards, and manuals for the core languages in this vault. Use these when the "distilled" wiki requires raw technical verification.

---

## 🏛️ C & C++ (Systems Core)

### [C] - ISO Standard
- **Official Drafts (C11, C17, C23):** [open-std.org/jtc1/sc22/wg14](https://www.open-std.org/jtc1/sc22/wg14/www/projects)
- **Primary Reference:** [cppreference.com/c](https://en.cppreference.com/w/c) (The industry standard community-maintained reference).

### [C++] - ISO Standard
- **Official Standard Committee:** [isocpp.org](https://isocpp.org/)
- **Definitive Technical Reference:** [cppreference.com/cpp](https://en.cppreference.com/w/cpp) (Exhaustive coverage of C++11 through C++23/26).
- **Core Guidelines:** [isocpp.github.io/CppCoreGuidelines](https://isocpp.github.io/CppCoreGuidelines/CppCoreGuidelines.html) (Bjarne Stroustrup's guide to writing "good" modern C++).
