# 📺 Remarkable Coding YouTube Channels

High-signal video resources focusing on systems engineering, low-level mechanics, terminal workflows, and code quality.

---

## 🏛️ Systems, Performance & Low-Level

### 🦀 Casey Muratori (Molly Rocket)
*   **YouTube**: [Casey Muratori](https://www.youtube.com/@cmuratori)
*   **Primary Stack**: C / C++ / Assembly
*   **Focus**: Low-level engine architecture, high-performance software engineering, and performance-aware programming. Home of *Handmade Hero* and *Computer Enhance*.

### 💻 Tsoding Daily (Alexey Kutepov)
*   **YouTube**: [Tsoding Daily](https://www.youtube.com/@tsodingdaily)
*   **Primary Stack**: C / Haskell / Assembly
*   **Focus**: Building compilers, game engines, and complex system utilities completely from scratch without frameworks. Deeply educational live-coding journeys.

### ⚙️ Nic Barker
*   **YouTube**: [Nic Barker](https://www.youtube.com/@NicBarker)
*   **Primary Stack**: C
*   **Focus**: Minimalist C programming, memory arena allocators, software styling, and game development fundamentals (famous for *"Tips for C Programming"*).
