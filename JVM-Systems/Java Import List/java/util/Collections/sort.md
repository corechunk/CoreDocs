# Method: sort(List<T> list)
## Class: `java.util.Collections`

Sorts the specified list into ascending order.

### Signature
- **Inputs:** `list: List` — The list to be sorted.
- **Returns:** `void`

### Example (Kotlin)
```kotlin
val list = mutableListOf(3, 1, 2)
Collections.sort(list)
```
