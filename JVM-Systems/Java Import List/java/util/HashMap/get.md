# Method: get(Object key)
## Class: `java.util.HashMap`

Returns the value to which the specified key is mapped, or null if this map contains no mapping for the key.

### Signature
- **Inputs:** `key: Object` — The key whose associated value is to be returned.
- **Returns:** `V` — The value to which the specified key is mapped, or null.

### Example (Kotlin)
```kotlin
val value = map.get("Key")
// or in Kotlin
val value = map["Key"]
```
