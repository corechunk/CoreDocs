# Method: put(K key, V value)
## Class: `java.util.HashMap`

Associates the specified value with the specified key in this map.

### Signature
- **Inputs:** 
    - `key: K` — Key with which the value is to be associated.
    - `value: V` — Value to be associated with the key.
- **Returns:** `V` — The previous value associated with the key, or `null`.

### Example (Kotlin)
```kotlin
val map = HashMap<String, Int>()
map.put("One", 1)
```
