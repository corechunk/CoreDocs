# Class: java.util.Scanner
## Purpose: User input reading

A simple text scanner which can parse primitive types and strings using regular expressions.

### Meaningful Methods

- [**nextLine()**](nextLine.md): Read a full line of text.
- [**nextInt()**](./Scanner/nextInt.md): Read an integer.
- [**hasNext()**](hasNext.md): Check if input is available.
