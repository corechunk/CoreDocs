# Class: java.util.Random
## Purpose: Random number generation

An instance of this class is used to generate a stream of pseudorandom numbers.

### Meaningful Methods

- [**nextInt()**](./Random/nextInt.md): Get random int.
- [**nextDouble()**](nextDouble.md): Get random double.
- [**nextBoolean()**](nextBoolean.md): Get random boolean.
