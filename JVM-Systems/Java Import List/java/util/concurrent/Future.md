# Class: java.util.concurrent.Future
## Purpose: Asynchronous Result

A `Future` represents the result of an asynchronous computation.

### Meaningful Methods

- [**get()**](./Future/get.md): Wait for and retrieve the result.
- [**isDone()**](./Future/isDone.md): Check if task is finished.
- [**cancel()**](./Future/cancel.md): Attempt to cancel task.
