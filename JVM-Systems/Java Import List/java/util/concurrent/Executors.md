# Class: java.util.concurrent.Executors
## Purpose: Thread Pool Factory

Static factory methods for creating different types of thread pools.

### Meaningful Methods

- [**newFixedThreadPool()**](newFixedThreadPool.md): Create pool with fixed threads.
- [**newSingleThreadExecutor()**](./Executors/newSingleThreadExecutor.md): Create pool with one thread.
- [**newCachedThreadPool()**](./Executors/newCachedThreadPool.md): Create pool that expands as needed.
