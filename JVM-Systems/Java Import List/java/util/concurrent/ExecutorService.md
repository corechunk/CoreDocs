# Class: java.util.concurrent.ExecutorService
## Purpose: Managing Thread Lifecycle

An `Executor` that provides methods to manage termination and methods that can produce a `Future` for tracking progress of one or more asynchronous tasks.

### Meaningful Methods

- [**submit()**](./ExecutorService/submit.md): Submit a task for execution.
- [**shutdown()**](./ExecutorService/shutdown.md): Initiates an orderly shutdown.
- [**isTerminated()**](./ExecutorService/isTerminated.md): Check if all tasks finished.
