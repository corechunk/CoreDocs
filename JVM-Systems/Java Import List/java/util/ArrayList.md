# Class: java.util.ArrayList

A resizable-array implementation of the `List` interface.

### Meaningful Methods

- [**add()**](./ArrayList/add.md): Add element to list.
- [**get()**](./ArrayList/get.md): Retrieve element by index.
- [**remove()**](remove.md): Delete element from list.
- [**size()**](size.md): Get number of elements.
