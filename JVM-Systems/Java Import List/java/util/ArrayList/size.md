# Method: size()
## Class: `java.util.ArrayList`

Returns the number of elements in this list.

### Signature
- **Inputs:** None
- **Returns:** `int` — The number of elements in this list.

### Description
Returns the actual number of items stored, not the current capacity of the internal array.

### Example (Kotlin)
```kotlin
val count = list.size()
// or in Kotlin
val count = list.size
```
