# Class: java.time.LocalTime
## Purpose: Time without Date

Represents a time (Hour, Minute, Second) in the ISO-8601 calendar system.

### Meaningful Methods

- [**now()**](./LocalTime/now.md): Get current time.
- [**of()**](./LocalTime/of.md): Create specific time.
