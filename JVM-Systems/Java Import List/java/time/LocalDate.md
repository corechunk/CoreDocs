# Class: java.time.LocalDate
## Purpose: Date without Time

Represents a date (Year, Month, Day) in the ISO-8601 calendar system.

### Meaningful Methods

- [**now()**](now.md): Get current date.
- [**of()**](./LocalDate/of.md): Create specific date.
- [**plusDays()**](./LocalDate/plusDays.md): Add days.
