# Method: toString()
## Class: `java.lang.Object`

Returns a string representation of the object.

### Signature
- **Inputs:** None
- **Returns:** `String` — A string representing the object.

### Description
By default, returns `ClassName@HashCode`. Usually overridden in subclasses to provide meaningful data.

### Example (Kotlin)
```kotlin
val obj = Object()
println(obj.toString())
```
