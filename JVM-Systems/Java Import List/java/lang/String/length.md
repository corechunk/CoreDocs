# Method: length()
## Class: `java.lang.String`

Returns the length of this string.

### Signature
- **Inputs:** None
- **Returns:** `int` — The number of characters in the string.

### Description
The length is equal to the number of Unicode code units in the string.

### Example (Kotlin)
```kotlin
val text = "Hello"
val len = text.length // 5
```
