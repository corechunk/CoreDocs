# Class: java.lang.String
## Purpose: Text representation

The most commonly used class in Java for handling text.

### Meaningful Methods

- [**substring()**](substring.md): Extract a portion of the string.
- [**length()**](length.md): Get character count.
- [**indexOf()**](indexOf.md): Find position of a character/string.
- [**replace()**](replace.md): Swap characters or sequences.
