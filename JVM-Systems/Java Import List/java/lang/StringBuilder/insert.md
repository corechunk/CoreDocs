# Method: insert(int offset, String str)
## Class: `java.lang.StringBuilder`

Inserts the string representation of the argument into this sequence.

### Signature
- **Inputs:** 
    - `offset: int` — The offset.
    - `str: String` — The string to insert.
- **Returns:** `StringBuilder` — A reference to this object.

### Example (Kotlin)
```kotlin
val sb = StringBuilder("Hlo")
sb.insert(1, "el") // "Hello"
```
