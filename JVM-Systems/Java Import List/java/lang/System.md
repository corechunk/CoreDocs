# Class: java.lang.System
## Purpose: System-level interactions

Provides access to standard input/output streams and system properties.

### Meaningful Methods

- [**currentTimeMillis()**](currentTimeMillis.md): Get current time in ms.
- [**out**](out.md): Standard output stream.
- [**getProperty()**](getProperty.md): Get system property.
