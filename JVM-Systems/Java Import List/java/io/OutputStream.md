# Class: java.io.OutputStream
## Purpose: Byte output

The abstract superclass of all classes representing an output stream of bytes.

### Meaningful Methods

- [**write(int b)**](./OutputStream/write.md): Write a byte.
- [**flush()**](flush.md): Flush the stream.
- [**close()**](./OutputStream/close.md): Close the stream.
