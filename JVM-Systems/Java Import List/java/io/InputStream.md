# Class: java.io.InputStream
## Purpose: Byte input

The abstract superclass of all classes representing an input stream of bytes.

### Meaningful Methods

- [**read()**](./InputStream/read.md): Read next byte.
- [**close()**](./InputStream/close.md): Close the stream.
- [**available()**](available.md): Estimate available bytes.
