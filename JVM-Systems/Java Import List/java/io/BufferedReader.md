# Class: java.io.BufferedReader
## Purpose: Efficient character reading

Reads text from a character-input stream, buffering characters so as to provide for the efficient reading of characters, arrays, and lines.

### Meaningful Methods

- [**readLine()**](readLine.md): Read a line of text.
- [**close()**](./BufferedReader/close.md): Close the stream.
- [**ready()**](ready.md): Check if stream is ready.
