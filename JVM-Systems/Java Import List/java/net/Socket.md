# Class: java.net.Socket
## Purpose: Client-side connection

Used by a client to connect to a server.

### Meaningful Methods

- [**getInputStream()**](./Socket/getInputStream.md): Get stream to read data.
- [**getOutputStream()**](./Socket/getOutputStream.md): Get stream to send data.
- [**close()**](./Socket/close.md): Terminate the connection.
- [**connect()**](./Socket/connect.md): Connect to a specific address.
