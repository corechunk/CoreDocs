# Class: java.net.ServerSocket
## Purpose: Server-side listening

Used by a server to wait for requests to come in over the network.

### Meaningful Methods

- [**accept()**](accept.md): Wait for and accept a client connection.
- [**close()**](./ServerSocket/close.md): Stop listening and close the server.
- [**getLocalPort()**](getLocalPort.md): Get the port being used.
