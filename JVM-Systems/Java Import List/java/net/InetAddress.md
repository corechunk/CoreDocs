# Class: java.net.InetAddress
## Purpose: IP Address Representation

This class represents an Internet Protocol (IP) address.

### Meaningful Methods

- [**getByName()**](getByName.md): Resolve hostname to IP.
- [**getLocalHost()**](./InetAddress/getLocalHost.md): Get local machine address.
- [**getHostAddress()**](./InetAddress/getHostAddress.md): Get IP as String.
