# Class: java.net.HttpURLConnection
## Purpose: Web Communication (HTTP)

Used to talk to REST APIs or download web content.

### Meaningful Methods

- [**setRequestMethod()**](setRequestMethod.md): Set GET, POST, etc.
- [**getResponseCode()**](getResponseCode.md): Get HTTP status code.
- [**getInputStream()**](./HttpURLConnection/getInputStream.md): Read response body.
- [**getOutputStream()**](./HttpURLConnection/getOutputStream.md): Send request body (for POST).
- [**setRequestProperty()**](setRequestProperty.md): Set HTTP headers.
