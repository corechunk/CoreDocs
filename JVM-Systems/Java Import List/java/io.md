# Package: java.io
## Input and Output

Handles reading and writing data to files, streams, and buffers.

### Meaningful Classes

- [**File**](File.md): File/Directory pathnames.
- [**InputStream**](InputStream.md): Byte input.
- [**OutputStream**](OutputStream.md): Byte output.
- [**BufferedReader**](BufferedReader.md): Efficient character reading.
- [**FileInputStream**](FileInputStream.md): Reading bytes from files.
- [**FileOutputStream**](FileOutputStream.md): Writing bytes to files.
