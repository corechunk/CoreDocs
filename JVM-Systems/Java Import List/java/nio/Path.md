# Class: java.nio.file.Path
## Purpose: File Path Representation

An object that may be used to locate a file in a file system. It is typically used with the `Files` class.

### Meaningful Methods

- [**of()**](of.md): Create a path from strings.
- [**toFile()**](./Path/toFile.md): Convert to a `java.io.File`.
- [**getParent()**](./Path/getParent.md): Get parent directory.
