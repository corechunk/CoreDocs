# Java Standard Library Structure

This is your master index for the Java Standard Library segments.

## [java.*](./java/Overview.md) (The Core)
Foundational packages for every application.
- [**lang**](./java/lang.md): Fundamentals (Strings, Math).
- [**util**](./java/util.md): Collections (Lists, Maps).
- [**io**](./java/io.md): Input/Output (Files, Streams).
- [**net**](./java/net.md): Networking (Sockets, HTTP, Servers).
- [**nio**](./java/nio.md): Modern File I/O.
- [**time**](./java/time.md): Modern Date and Time API.
- [**math**](./java/math.md): Arbitrary-precision math.

## [javax.*](./javax/Overview.md) (Standard Extensions)
Extensions for desktop UI and enterprise features.
- [**swing**](./javax/swing.md): Desktop GUIs.
- [**crypto**](./javax/crypto.md): Encryption.
- [**ssl**](./javax/ssl.md): Secure connection management.

## [jdk.*](./jdk/Overview.md) (JDK Tools)
Internal tools and monitoring.
- [**compiler**](./jdk/compiler.md): Access to `javac`.
- [**jfr**](./jdk/jfr.md): Performance profiling.
- [**jshell**](./jdk/jshell.md): Snippet execution.
