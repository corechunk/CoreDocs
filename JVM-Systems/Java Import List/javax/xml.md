# Segment: javax.xml
## XML Processing

Standard APIs for processing XML documents (DOM, SAX, StAX).

### Key Classes

| Class | Purpose |
| :--- | :--- |
| **DocumentBuilder** | Define the API to obtain DOM Document instances from an XML document. |
| **SAXParser** | Defines the API that wraps a SAX (Simple API for XML) reader. |
| **Transformer** | An instance of this abstract class can transform a source tree into a result tree. |
