# Segment: jdk.jfr
## Java Flight Recorder

A tool for collecting diagnostic and profiling data about a running Java application.

### Key Classes

| Class | Purpose |
| :--- | :--- |
| **FlightRecorder** | The central point for managing recordings and events. |
| **Recording** | Represents a recording of events. |
| **Event** | Base class for JFR events. |
