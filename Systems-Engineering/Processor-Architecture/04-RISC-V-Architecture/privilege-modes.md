# 🛡️ RISC-V Privilege Modes (U, S, M)

## 1. Privilege Level Architecture
- **User Mode (U-Mode):** Unprivileged application execution.
- **Supervisor Mode (S-Mode):** Privileged operating system kernel execution (managing page tables and virtual memory).
- **Machine Mode (M-Mode):** Highest privileged mode with direct, uninhibited hardware access (executing OpenSBI or custom boot firmware).
