## 2. Visual Logic
### The "Odd Degree" Rule
- **Eulerian Circuit:** All nodes must have an **Even** degree.
- **Eulerian Path:** Exactly zero or two nodes must have an **Odd** degree.

---

## 3. The "Aha!" Moment
This is a counting problem. You don't need to find the path; just count the degrees of all nodes to know if it's possible.
