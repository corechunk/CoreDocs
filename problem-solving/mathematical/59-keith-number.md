# 59 | Keith Number

A number that appears in a Fibonacci-like sequence generated by its own digits.

---

## 1. The Objective
A number $n$ with $d$ digits is a **Keith Number** if it appears in a sequence where:
- The first $d$ terms are the digits of $n$.
- Every subsequent term is the sum of the previous $d$ terms.

- **Example:** $197$ (3 digits)
- Sequence: **1, 9, 7**, 17, 33, 57, 107, **197**. (Keith!)

---

## 2. Visual Logic
### The "Sliding Sum" (n=14)
1. Digits: 1, 4.
2. Next: $1 + 4 = 5$. (Seq: 1, 4, 5)
3. Next: $4 + 5 = 9$. (Seq: 1, 4, 5, 9)
4. Next: $5 + 9 = 14$. (Seq: 1, 4, 5, 9, 14) -> **YES**

---

## 3. The "Aha!" Moment
You must keep a "window" of the last $d$ terms. Once the sum equals $n$, it's a Keith number. If the sum exceeds $n$ without hitting it, it's not.

## 4. C++ Implementation

```cpp
#include <iostream>
#include <vector>
#include <numeric>

/**
 * Strategy: Dynamic Window Fibonacci
 */
bool isKeith(int n) {
    std::vector<int> digits;
    int temp = n;
    while (temp > 0) {
        digits.insert(digits.begin(), temp % 10);
        temp /= 10;
    }
    
    int d = digits.size();
    std::vector<long long> seq = digits;
    
    long long nextSum = 0;
    while (nextSum < n) {
        nextSum = 0;
        // Sum the last 'd' elements
        for (int i = seq.size() - d; i < seq.size(); i++) {
            nextSum += seq[i];
        }
        
        if (nextSum == n) return true;
        seq.push_back(nextSum);
    }
    
    return false;
}

int main() {
    int num = 197;
    if (isKeith(num)) std::cout << num << " is a Keith number." << std::endl;
    else std::cout << num << " is NOT a Keith number." << std::endl;
    return 0;
}
```
