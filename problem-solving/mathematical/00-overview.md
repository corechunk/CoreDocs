# 🧭 Mathematical Mastery: 100-Problem Roadmap

Welcome to the 100-problem mathematical challenge. This roadmap is designed to take you from basic iterative logic to advanced algorithmic strategy using C++.

## 🛡️ The Learning System
Every problem in this vault includes:
1. **Objective:** What are we solving?
2. **Visual Logic:** How do the numbers move? (ASCII/Tracing).
3. **Aha! Moment:** The core insight.
4. **20-Line Gap:** Mental space before the solution.
5. **Multiple Strategies:** Iterative, Recursive, and Optimized versions where applicable.
