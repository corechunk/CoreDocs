## 🚀 Strategy
Every problem in this vault follows a strict pedagogical structure:
1. **Objective:** What are we solving?
2. **Visual Logic:** Mental model and diagrams.
3. **Technical Rationale:** Why this approach?
4. **Implementation:** Optimized C++ solutions.
